Script Tags Demo

This example demonstrates traditional script tag usage with the UMD build for direct browser use.

Features:

  • Uses global L object
  • Works without build tools
  • CDN-ready for quick prototyping
  • Programmatic control API

Try it: Use the fullscreen button (top-left of the map) to enter/exit fullscreen mode.

Programmatic API:

You can also control fullscreen via JavaScript using map.fullscreenControl.toggleFullScreen()
